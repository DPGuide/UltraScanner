***Step 1: Turn Python into an .exe (with PyInstaller)
There's a brilliant tool that compresses your entire Python script, including all libraries (PyQt, NumPy, etc.), into a single Windows file.
Open your terminal (command prompt/CMD) in the location of your Python file.
Install the compression program once:
pip install pyinstaller
Now enter this magic command (replace cosmic.py with the actual name of your file):
pyinstaller --onefile --noconsole cosmic.py
What happens now?
Your PC will work for one to two minutes. After that, you'll find a new subfolder called dist in your directory.
Inside is a brand new cosmic.exe file. This is your complete Python program, ready to run on any Windows PC!***
